# Linux 桌面悬浮任务列表

这是一个基于 **Python Tkinter** 的轻量级桌面任务列表应用，适用于 Linux 桌面环境。

## 功能

- 桌面窗口置顶悬浮（Always on top）
- 新增、删除、切换任务完成状态
- 任务优先级：高 / 中 / 低
- 为每个任务设置到期时间（`YYYY-MM-DD HH:MM`）
- 支持提醒通知：到期时 / 提前 5、15、30、60 分钟
- 支持 Linux 开机自启（应用内一键启用 / 关闭）
- 自动按“未完成优先、优先级高优先、到期时间更早优先”排序
- 支持“全部 / 未完成 / 已完成”筛选
- 任务自动保存到本地 `tasks.json`

## 运行方式

要求：Linux 系统安装 Python 3，并带有 Tkinter。

```bash
python3 app.py
```

## 使用说明

1. 在输入框中输入任务名称。
2. 选择任务优先级（高 / 中 / 低）。
3. 如需截止控制，填写到期时间，格式为 `YYYY-MM-DD HH:MM`。
4. 如需通知提醒，选择提醒时间（需要先填写到期时间）。
5. 点击“添加任务”。
6. 双击任务行或点击“切换完成”可以切换完成状态。
7. 点击“重置提醒”可让任务在下一个提醒时间再次弹窗。
8. 点击“删除任务”可移除任务。
9. 点击“启用开机自启”可写入 `~/.config/autostart/task-list.desktop`。
10. 点击“关闭开机自启”可移除对应自启文件。
11. 默认窗口置顶，可点击“窗口置顶: 开”切换。

## 提醒机制说明

- 程序每 30 秒检查一次任务是否达到提醒时间。
- 达到提醒时间后会弹出提醒框并响铃一次。
- 已提醒过的任务不会重复提醒，除非你手动点击“重置提醒”或把任务恢复为未完成。
- 若任务已过期但尚未完成，会显示为“已逾期”。

## 开机自启说明

- 程序使用 Linux 桌面环境常见的 `~/.config/autostart/` 机制。
- 启用后会生成 `task-list.desktop`，内容会指向当前 Python 解释器和当前项目下的 `app.py`。
- 关闭后会删除该 `.desktop` 文件。
- 该方式通常适用于 GNOME、KDE、XFCE 等桌面环境。

## 数据存储

任务数据保存在程序同目录下的 `tasks.json` 中，示例格式：

```json
[
  {
    "title": "完成日报",
    "priority": "高",
    "due_at": "2026-03-18 18:00",
    "reminder_label": "提前15分钟",
    "completed": false,
    "reminded": false
  }
]
```
